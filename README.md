# WeatherView
날씨에 따라 다양한 애니메이션을 보여줄 수 있는 커스텀 애니메이션 뷰를 제작하였습니다.

### Provide Features
#### WeatherKind
- Snow
- Rain

#### WeatherFalling
- fallingItemCount
- fallingItemSpeed
- fallingItemAngle
- fallingItemRotate
- fallingItemColor
- fallingItemImage
- fallingItemFromTheSky
